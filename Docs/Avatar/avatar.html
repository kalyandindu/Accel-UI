Avatar

Avatar component is used to display images, initials etc. that represent the user. It comes in various sizes from x-small to x-large.

Rounded Avatar

Most commonly used avatar for profile picture. Use class avatar-round to round the container and avatar_img for image to fit into container

Squared Avatar:

This is a squared avatar with rounded corners. This typically can also be used as profile picture. Use class avatar-squared for rounded square container and avatar_img for image to fit into container.
