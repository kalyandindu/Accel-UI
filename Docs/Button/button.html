Buttons

Button component is used to trigger an event that it is associated with. Example being submition of form, Action on a modal/card etc.

Primary button:

To be used for call to action buttons. Use btn-primary along with btn class for primary button. Similarly btn-outline and btn-disabled for outline and disabled buttons.

Link

Display of link in the form of button. Use btn-link class for the same.

link 
          
            link

Button with Icon:

Wrapping icon with button content. Use btn-wrapper class to achieve this.

Fixed Button:

Include btn-float class to make any button floatable.
