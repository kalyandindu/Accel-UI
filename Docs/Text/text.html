Text Utilities

Helper classes for header and paragraphs. This overwrites the existing fontsize, line-height and font-weight of h1 to h4 headers and paragraph. This uses classes Heading-{number} and p-{size} where number ranging from 1 to 4 and size being either of sm, md or lg.
